# Têmpera - Escola de Artes

Site estático da Escola de Artes Têmpera.

## Deploy no Azure

Este site está hospedado no Azure App Service.

### Tecnologias
- HTML5
- CSS3
- JavaScript (se aplicável)

### Estrutura
- `index.html` - Página principal
- `images/` - Imagens do site

## Instruções de Deployment

1. Clone o repositório
2. Faça o login no Azure CLI: `az login`
3. O site será deployado automaticamente via GitHub Actions ou Azure DevOps
